# 谈设计
```赋予设计以内涵，设计才能称之为设计。而不算设计的话，造轮子的意义也就不大了。```
<br/>
我们要创造一个怎么的starblog（三人共一个博客）？
这个问题一时半会儿还真说不清楚。
starblog的诞生源于学习经验的分享，想着能让Alray分享一些他平常学到的小知识，看看他是怎么自学到这种地步的。
<br/>
既然如此，starblog的 **核心** 就是 **经验分享**，任何其他的附加东西都不能影响了这个主核心。
了解了**核心**，那么设计也就有了着重，不能因为要个性而影响了重要的**阅读**功能。
<br/>
对于我们（作者）来说，我们要知道我们自己要写什么样的内容，怎么来写，怎么更加清晰的把**经验**传达出去。
<br/>
关于读者，读者可能不仅仅是我们自己，还会有我们的其他朋友，外来的访客。
我（Lai追求）个人觉得，除了博客的内容本身，还需要有一个良好的阅读体验，让人眼前一亮的各种个性化有趣的设计。
说到这设计，我就不得不提Alray做的iwanna关卡，一个闯关游戏的关卡按常理来说就是各种难度的闯就行了，但是Alray做的关卡却带来了许多的“**艺术性**”，
游戏开始的动画、自己新做的死亡动画、乘坐“火箭”…………很多超乎我想象的东西，这才是设计啊！
这就是我所追求的设计————画风温和、眼前一亮、趣而不乏、宣扬个性。
### 关于阅读的小想法
能不能给阅读的文字做上动效或者设置特殊的排版，一句话一句话，这样能让阅读有了一种新奇的体验，相信注意力也能更加集中，滚动滚动条也可以像拖动进度条那样调整阅读进度。
遇到重点的时候，还会进行放大或者变色处理，各种名词或者其他什么的注解也可以点击查看详细。
